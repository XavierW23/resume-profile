# Case Study: Enterprise Identity Automation

## Summary

At Chevron, I built Microsoft Graph and PowerShell automation to improve enterprise identity hygiene and cloud compliance.

## Problem

Large enterprise identity environments accumulate stale objects and manual operational tasks. Those stale objects can create compliance noise, operational risk, and avoidable engineering toil.

## Solution

I designed automation using Microsoft Graph, PowerShell, and Azure DevOps pipelines to identify and reduce stale Microsoft Entra ID objects while supporting enterprise cloud identity operations.

## Key Engineering Work

- Built Microsoft Graph automation for identity object cleanup.
- Reduced more than 5,000 stale Microsoft Entra ID objects.
- Used Azure DevOps pipelines to standardize execution.
- Supported Conditional Access improvements using phishing-resistant MFA and compliant device enforcement.
- Collaborated with security and platform teams supporting thousands of users.

## Results

- Improved enterprise identity compliance.
- Reduced manual administrative effort.
- Strengthened repeatability and operational reliability in cloud identity workflows.
