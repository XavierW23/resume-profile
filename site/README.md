# Xavier White Portfolio Website

Interview-ready portfolio website for Xavier White.

## Stack

- React
- vinext
- Tailwind CSS
- Cloudflare Workers-compatible Sites build

## Local Development

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
```

## Content

The site presents Xavier as a software engineer specializing in autonomous AI systems, with project sections for Agency OS, Runtime Supervisor, AI Market Intelligence, and Enterprise Identity Automation.

Agency OS remains proprietary. The site uses public-safe architecture diagrams and descriptions.
