## Publishing Checklist

- Create each GitHub repository as public.
- Copy each folder's contents into its matching repository.
- Keep Agency OS private.
- Do not include real customer data, production prompts, credentials, API keys, Make webhook URLs, Slack tokens, or proprietary business rules.
- Add screenshots only if they are sanitized and do not reveal client or business-sensitive data.
